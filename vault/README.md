# Vault Map

This vault uses the PARA method:

- **Projects/** — things with a specific goal and an end date (tied to knowledge/PROJECTS.md)
- **Areas/** — ongoing responsibilities with no end date (health, finances, this Jarvis system itself)
- **Resources/** — reference material and topics of interest, not tied to a specific project
- **Archive/** — inactive items from any of the above, kept for reference

When creating a new note, put it in the right folder based on this definition rather than defaulting to the vault root.
