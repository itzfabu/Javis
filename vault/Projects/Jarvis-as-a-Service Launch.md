# Jarvis-as-a-Service Launch

## Summary

Turn the Jarvis personal AI assistant system into income — either a done-for-you custom build for other individuals/small businesses, or a demo-driven offering that generates bespoke consulting leads. Tracked in knowledge/PROJECTS.md, owner agent: project-manager. Linked to the "Monetize the Jarvis System" goal (knowledge/GOALS.md, quarterly, active).

## Key details

- **Pricing/positioning (business-analyst):** don't list this as a marketplace gig — "Claude Code setup" is commoditized on Fiverr at $15–$45. Better positioned as a demo-driven lead magnet for bespoke consulting sourced outside Fiverr/Upwork, priced in the $500–$2,000 build / $2,000+ retainer range once productized. Full findings: [[Jarvis-as-a-Service Pricing Research]].
- **Demo site (webdesigner):** the real orb dashboard (`C:\Jarvis\orb`) can't be safely embedded live — it's wired to Fabio's real local Claude Code session with a hardcoded token. v1 plan: a static single-page site built around a screen recording of the real dashboard as the proof asset, with a sandboxed fake-data replica as a later "try it yourself" section.
- **Current status:** scoping/research phase — no build work started yet. Next concrete step is recording a demo video of the orb + agent delegation in action.
